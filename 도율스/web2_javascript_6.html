데이터타입 문자열과 숫자를 알아보자

javascript에는 6개의 데이터 타입과 1개의 객체가 있다.

  1. Boolean
  2. Null
  3. Undefined
  4. Number
  5. String
  6. Symbol
  7. and Object

익숙한 데이터타입인 문자열(String)과 숫자(Number)를 파악해보자


숫자

alert(1); 1이라는 경고창을 생성함

alert(1+1); 1+1이라는 연산자를 통해 2라는 값을 지정하여 경고창을 생성함

산술 연산자 종류

  • +
  • -
  • *
  • /

문자열

"". ''를 통해 문자로 표현

문자열의 속성값

  • .length(): 문자열의 글자가 몇 글자인지 확인이 가능
  • .toUpperCase(): 소문자를 대문자로 표현
  • .indexOf('o')=4: 문자열의 해당하는 순서 찾기
  • .trim(): 공백을 삭제

정리하자면

1+1은 숫자로 표현되며 2라는 값을 나오게 하지만

"1"+"1"은 문자값으로 표현되며 "11"라는 값을 나오게 한다.

즉 데이터 타입을 정확하게 표현해야 함


Tip

java에서 문자열은 string=Properties 와 같다.
