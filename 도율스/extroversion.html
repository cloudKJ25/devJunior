npm설치 방법

  1. 구글에 node js를 검색한다.
  2. 14.15.1 버전을 다운로드 받는다.
  3. 소스트리를 통해 https://github.com/kwangwoonInfoCont/kwinfocont.git 이 주소와 연동한다.
  4. 소스트리 탐색기를 통해 PowerShall을 관리자 권한으로 실행한다.
  5. 만약 오류가 뜬다면 cmd를 관리자 권한을 통해 실행한다. (cd..)은 상위 폴터로 이동 (cd 명)은 해당 폴더로 이동한다.
  6. npm install
  7. npm i vue-template-compiler -save / 이 내용은 컴퓨터 마다 다를 수 있음
  8. npm run serve
  9. 순서대로 명령어를 입력한다.
  10. 설치가 안료되면 npm설치 끝~
